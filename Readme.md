* **Tasks:**

  * Task 1: Select a paragraph using getElementById and change its text content when a button is clicked.
  * Task 2: Select multiple list items using querySelectorAll and change their color when a button is clicked.
  * Task 3: Select an element using querySelector and update its innerHTML when a button is clicked.
  * Task 4: Change the background color of a div when a button is clicked.
  * Task 5: Toggle a CSS class on a div to change its style when a button is clicked.
  * Task 6: Add a new list item (li) to an existing ul when a button is clicked.
  * Task 7: Remove the last list item (li) from an existing ul when a button is clicked.
  * Task 8: Create a counter with Increase and Decrease buttons.
  * Task 9: Create a simple form. When submitted, display the input value on the page without reloading.
  * Task 10: Toggle the page background color only when clicking inside this section.
